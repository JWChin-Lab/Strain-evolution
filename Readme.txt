Automated mutagenesis.sproj - SAMI software import file (scheduling/pipeline software). 
AutomatedMutagenesisODCheckIndexed.bmf - Biomek method, run inside SAMI method, contains VB script for analysing OD600 data. 
Deck layout.imp - Biomek import file of Biomek FXp deck layout

VB_1.txt - Embedded script in Biomek method for identifying wells that exceed target OD
VB_2.txt - Embedded script in Biomek method for calculating estimated time required for further incubation to reach target OD

This method was written by Wolfgang H. Schmied (original version: 10.1038/s41586-018-0773-z), with modifications by Kim C. Liu.
